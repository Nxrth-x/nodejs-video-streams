# NodeJS video streams

A simple test on streaming videos with NodeJS

## Preview

![Video stream preview with the video of a cat](public/thumbnails/preview.jpeg)

## Installation

1. Install dependencies

```sh
$ yarn
```

2. Start the server

```sh
$ yarn dev
```
